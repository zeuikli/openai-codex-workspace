## 優點（R2 相對 BEFORE 的真增量）

1. **治理自應用閉環**：advisory vs hard-block、hooks≠baseline、packet_absent、open backlog「結案前禁止的宣稱」——不只寫給 agent，也寫給**文件作者自己**。
2. **採用路徑可走完**：ADOPTION 決策樹 + 四欄對照 + 禁止清單，直接回應「外部會把 v4 當已 enforce」的 C1 誤用面。
3. **詞彙單頁降低常駐稅**：GLOSSARY describe-only、衝突以 CORE 為準——清晰度最大受益項。
4. **EXPORT 可建且可稽核**：preamble 源檔 + build hash，治理層不再靠手改 artifact 漂移。
5. **Ledger 誠實開帳**：#13-packet / F18-action 從「散落 caveat」升級為可追蹤 ID——反腐蝕可操作。

## 殘餘風險（分數為何停在 8.5）

1. **Body 覆蓋率未質變**：SPEC §2.2 大量 `empty`/`advisory`；可執行 7.4 已是 guides 上限附近。
2. **紅軸仍紅且未測 hooks 效益**：lexical 可繞；#2c 不做則 P0-3 永遠 PARTIAL。
3. **證據鏈張力未消**：刪 packet + 演化吃 full trace = 已知自相矛盾；stub 承認 ≠ 修復。
4. **可攜實證缺口**：#14 不結，adapter 再漂亮也是 uncalibrated 天花板。
5. **n=1 文化債**：呈現修了，換代/宣傳若仍引用 20/22 當穩定 verdict = 治理回潮。
